# 歡迎來到 機器感知與邊緣運算實驗室 
這裡存放了實驗室的所有運作規範與資源，請每位成員務必詳閱。

## 實驗室風氣

本實驗室期望每位成員都能在研究過程中，培養紮實的專業能力與嚴謹的學術態度，成為AI浪潮下能真正理解技術、善用工具並持續成長的人。
實驗室重視學術誠信與研究品質。
所有研究、實驗與報告皆應秉持認真求實的態度，確保資料來源清楚、實驗數據真實可靠，切勿造假、隱匿問題，或在未經查證的情況下全然依賴生成式 AI 的產出。
指導老師願意給予學生時間探索與成長，但研究進度終究是自己的責任。
成員應培養自主學習的習慣，不應只是等待指令，而應主動閱讀文獻、嘗試方法、整理問題，並尋找可能的解決方案。
實驗室的設備、公物與資源多屬校方財產，未來也會有其他成員持續使用。
請大家愛惜使用、妥善維護，並共同維持一個安全、整潔且適合研究的環境。
