# 畢業規範

本實驗室畢業流程除須符合系所與學校規定外，亦須完成實驗室之研究與交接要求。

---

## 畢業流程

1. **滿足系所規定**  
   - 修課學分與成績達標  
   - 符合系上畢業條件（如：論文發表、修業年限等）  

2. **完成實驗室研究發表**  
   - 至少參與並協助完成一篇期刊或研討會論文  
   - 須對研究內容具備實質貢獻（非僅掛名）  

3. **論文口試與修訂**  
   - 通過論文口試  
   - 依委員意見完成論文修改與最終定稿  

4. **實驗室研究交接**  
   須完成以下交接事項：
   - 研究程式整理（GitHub Repository）
   - README 文件（包含環境、執行方式、資料來源）
   - 投影片與論文相關資料歸檔  
   - 與接手成員進行實際操作教學  

5. **完成校方行政流程**  
   - 依系辦與學校時程完成所有畢業手續  
   - 包含：離校申請、圖書館繳交、系辦簽核等  

---

## 交接完成標準

未達以下條件者，視為未完成交接：(詳參行政事務與報帳流程>實驗室離開前業務交接)
- Repository 可成功 Clone 並執行  
- README 說明完整且清楚  
- 接手者可在新環境成功操作  
- 論文與投影片皆已整理並歸檔  

---

## 補充說明

- 畢業流程須依序完成  
- 若未完成實驗室規範，指導教授有權暫緩相關簽核流程  
- 所有資料須確保可重現性與可理解性
